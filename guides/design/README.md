# Design System Documentation

This directory contains all design system guides and UI/UX documentation for the Open Overheid platform.

## 📚 Design Guides

### Material Design 3
**File:** `materialdesign_llm.md`

Comprehensive guide for implementing Material Design 3 principles:
- Color tokens and theming
- Typography scale
- Spacing system (8dp grid)
- Component patterns
- Dark mode support

### Tailwind CSS
**File:** `tailwind_llm.md`

Tailwind CSS configuration and usage guide:
- Custom theme configuration
- Utility class patterns
- Component styling
- Responsive design patterns

### WCAG Accessibility
**File:** `wcag_llm.md`

WCAG 2.2 AA compliance guidelines:
- Accessibility requirements
- ARIA attributes
- Keyboard navigation
- Screen reader support
- Focus management

### Font Awesome Icons
**File:** `fa_llm.txt`

Font Awesome icon usage guide:
- Icon selection guidelines
- WCAG-compliant icon implementation
- Icon accessibility patterns
- Icon variants (sharp-duotone, light)

### TOOI Specifications
**File:** `tooi_llm.md`

The Open Overheid Index (TOOI) specifications and requirements.

## 🎨 Current Implementation

The platform uses:
- **Material Design 3** for design principles
- **Tailwind CSS v4** for styling
- **Flux UI Theme** (blue accent, neutral base)
- **Font Awesome Free 6.5.2** for icons
- **WCAG 2.2 AA** compliance standards
