# TipTap PHP/Laravel Dokümantasyonu

Bu klasör TipTap editor'ün PHP ve Laravel ile kullanımı için hazırlanmış dokümantasyonu içerir.

## Dosyalar

- `tiptap-php-documentation.md` - Web scraping ile çekilmiş resmi dokümantasyon
- `tiptap-php-laravel-guide.md` - Laravel entegrasyonu için detaylı rehber
- `scraper.py` - Dokümantasyonu çekmek için kullanılan Python scraper

## Kaynak
